## Welcome to Mintlify

Mintlify is a modern documentation platform that combines powerful features with an elegant design. Build beautiful documentation sites that your users will love, with AI-powered search, automatic deployments, and a seamless editing experience.

## How it works

Mintlify uses a docs-as-code approach to manage your documentation. Every page on your site has a corresponding MDX file stored in your documentation repository.

<Steps>
  <Step title="Connect your repository">
    Link your GitHub repository to Mintlify. Your documentation files live alongside your code.
  </Step>
  
  <Step title="Write and edit">
    Create and update documentation files using the web editor or your preferred IDE locally.
  </Step>
  
  <Step title="Deploy automatically">
    Push changes to your repository and Mintlify automatically builds and deploys your site.
  </Step>
</Steps>
