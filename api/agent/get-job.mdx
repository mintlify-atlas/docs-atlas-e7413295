---
title: "Get Agent Job"
api: "GET https://api.mintlify.com/v1/agent/{projectId}/job/{id}"
description: "Retrieve the details and status of a specific agent job"
keywords: ["agent job", "status", "retrieve", "details"]
---

This endpoint retrieves the details and status of a specific agent job by its unique identifier. Use this to check the progress, status, and results of a previously created agent job.

## Authentication

Requires an admin API key (prefixed with `mint_`). Generate one on the [API keys page](https://dashboard.mintlify.com/settings/organization/api-keys) in your dashboard.

## Path Parameters

<ParamField path="projectId" type="string" required>
  Your project ID. Can be copied from the [API keys](https://dashboard.mintlify.com/settings/organization/api-keys) page in your dashboard.
</ParamField>

<ParamField path="id" type="string" required>
  The unique identifier of the agent job to retrieve.
</ParamField>

## Response

The response includes information such as:
- Job execution status and completion state
- Branch information and pull request details
- Session metadata and timestamps

<ResponseField name="sessionId" type="string">
  The unique session identifier.
</ResponseField>

<ResponseField name="subdomain" type="string">
  The subdomain this session belongs to.
</ResponseField>

<ResponseField name="branch" type="string" nullable>
  Git branch name where changes were made.
</ResponseField>

<ResponseField name="haulted" type="boolean">
  Whether the session execution was halted.
</ResponseField>

<ResponseField name="haultReason" type="string">
  Reason for session halt. Possible values: `completed`, `github_missconfigured`, `error`.
</ResponseField>

<ResponseField name="pullRequestLink" type="string">
  Link to the created pull request.
</ResponseField>

<ResponseField name="messageToUser" type="string">
  Message for the user about the session outcome.
</ResponseField>

<ResponseField name="todos" type="array">
  List of todo items from the session.
  
  <Expandable title="Todo object">
    <ResponseField name="content" type="string">
      Brief description of the task.
    </ResponseField>
    
    <ResponseField name="status" type="string">
      Current status of the task. Possible values: `pending`, `in_progress`, `completed`, `cancelled`.
    </ResponseField>
    
    <ResponseField name="priority" type="string">
      Priority level of the task. Possible values: `high`, `medium`, `low`.
    </ResponseField>
    
    <ResponseField name="id" type="string">
      Unique identifier for the todo item.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="createdAt" type="string">
  Timestamp when the session was created (ISO 8601 format).
</ResponseField>
