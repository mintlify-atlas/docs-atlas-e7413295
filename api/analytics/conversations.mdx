Use this endpoint to export AI assistant conversation history from your documentation. Each conversation includes the user query, assistant response, sources cited, and query category.

Paginate through results using the `cursor` parameter returned in the response. Continue fetching while `hasMore` is `true`.

## Authentication

Requires an admin API key (prefixed with `mint_`). Generate one on the [API keys page](https://dashboard.mintlify.com/settings/organization/api-keys) in your dashboard.

## Path Parameters

<ParamField path="projectId" type="string" required>
  Your project ID. Can be copied from the [API keys](https://dashboard.mintlify.com/settings/organization/api-keys) page in your dashboard.
</ParamField>

## Query Parameters

<ParamField query="dateFrom" type="string">
  Date in ISO 8601 or YYYY-MM-DD format. Filter results to include conversations from this date onwards.
  
  Example: `2024-01-01`
</ParamField>

<ParamField query="dateTo" type="string">
  Date in ISO 8601 or YYYY-MM-DD format. `dateTo` is an exclusive upper limit. Results include dates before, but not on, the specified date.
  
  Example: `2024-01-31`
</ParamField>

<ParamField query="limit" type="number" default={100}>
  Max results per page. Minimum: 1, Maximum: 1000.
</ParamField>

<ParamField query="cursor" type="string">
  Pagination cursor (ULID format) from the previous response's `nextCursor` field.
</ParamField>

## Response

<ResponseField name="conversations" type="array">
  List of assistant conversations.
  
  <Expandable title="Conversation object">
    <ResponseField name="id" type="string">
      Unique conversation identifier.
    </ResponseField>
    
    <ResponseField name="timestamp" type="string">
      Timestamp when the conversation occurred.
    </ResponseField>
    
    <ResponseField name="query" type="string">
      The user's question to the assistant.
    </ResponseField>
    
    <ResponseField name="response" type="string">
      The assistant's response.
    </ResponseField>
    
    <ResponseField name="sources" type="array">
      Documentation pages referenced in the response.
      
      <Expandable title="Source object">
        <ResponseField name="title" type="string">
          Title of the referenced documentation page.
        </ResponseField>
        
        <ResponseField name="url" type="string">
          URL of the referenced documentation page.
        </ResponseField>
      </Expandable>
    </ResponseField>
    
    <ResponseField name="queryCategory" type="string" nullable>
      Auto-assigned category grouping for the conversation, if applicable.
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="nextCursor" type="string" nullable>
  Cursor to retrieve the next page of results. Null if no more results.
</ResponseField>

<ResponseField name="hasMore" type="boolean">
  Whether additional results are available beyond this page.
</ResponseField>

## Filtering

Filter conversations by date range using `dateFrom` and `dateTo` parameters.

## Conversation data

Each conversation includes:
- **query**: The user's question
- **response**: The assistant's answer
- **sources**: Pages referenced in the response, with title and URL
- **queryCategory**: Classification of the query type (if available)

<RequestExample>

```bash cURL
curl -X GET "https://api.mintlify.com/v1/analytics/your_project_id/assistant?dateFrom=2024-01-01&limit=100" \
  -H "Authorization: Bearer mint_your_api_key_here"
```

```javascript JavaScript
const response = await fetch(
  'https://api.mintlify.com/v1/analytics/your_project_id/assistant?dateFrom=2024-01-01&limit=100',
  {
    method: 'GET',
    headers: {
      'Authorization': 'Bearer mint_your_api_key_here',
    },
  }
);

const data = await response.json();
console.log(data.conversations);

// Paginate through results
let cursor = data.nextCursor;
while (cursor && data.hasMore) {
  const nextPage = await fetch(
    `https://api.mintlify.com/v1/analytics/your_project_id/assistant?cursor=${cursor}`,
    {
      method: 'GET',
      headers: { 'Authorization': 'Bearer mint_your_api_key_here' },
    }
  );
  const pageData = await nextPage.json();
  console.log(pageData.conversations);
  cursor = pageData.nextCursor;
}
```

```python Python
import requests

response = requests.get(
    'https://api.mintlify.com/v1/analytics/your_project_id/assistant',
    headers={'Authorization': 'Bearer mint_your_api_key_here'},
    params={
        'dateFrom': '2024-01-01',
        'limit': 100
    }
)

data = response.json()
print(data['conversations'])

# Paginate through results
cursor = data['nextCursor']
while cursor and data['hasMore']:
    response = requests.get(
        'https://api.mintlify.com/v1/analytics/your_project_id/assistant',
        headers={'Authorization': 'Bearer mint_your_api_key_here'},
        params={'cursor': cursor}
    )
    data = response.json()
    print(data['conversations'])
    cursor = data['nextCursor']
```
