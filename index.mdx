<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white">
    Quick start
  </h2>
  <p className="mt-2 text-base text-gray-600 dark:text-gray-400">
    Get your documentation site live in minutes
  </p>
  
  <Steps className="mt-8">
    <Step title="Install the Mintlify CLI">
      Install the CLI to preview and develop your docs locally.
      
      ```bash npm
      npm install -g mint
      ```
      
      ```bash yarn
      yarn global add mint
      ```
      
      ```bash pnpm
      pnpm add -g mint
      ```
    </Step>
    
    <Step title="Create your docs repository">
      Clone the starter template or connect your existing repository.
      
      ```bash
      git clone https://github.com/mintlify/starter.git
      cd starter
      ```
      
      <Note>
        You can also use the Mintlify dashboard to create a new repository automatically.
      </Note>
    </Step>
    
    <Step title="Preview your docs">
      Run the development server to see your docs in real-time.
      
      ```bash
      mint dev
      ```
      
      Your documentation will be available at `http://localhost:3000`.
    </Step>
    
    <Step title="Deploy to production">
      Connect your repository to Mintlify and deploy with a single click.
      
      <Tip>
        Every push to your main branch automatically deploys to production. Preview deployments are created for pull requests.
      </Tip>
    </Step>
  </Steps>
</div>
